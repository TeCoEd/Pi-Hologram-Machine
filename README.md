# Pi-Hologram-Machine
Displays Holograms using a Raspberry Pi

# What you need
*IKEA table
*Old monitor
*3 x buttons
*Raspberry Pi 3 A+ (Or other model)
*Jumper jerky
*Clear Perspex Pyramid
